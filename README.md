# Context Dictionary

A browser extension for language learners that enables contextual vocabulary collection with AI-powered story generation and spaced repetition study mode.

**Website**: [https://www.context-dictionary.com/](https://www.context-dictionary.com/)

## Features

- **Context-first word saving**: Save words from web pages with their original context
- **AI story generation**: Generate engaging stories using your vocabulary with Gemini, OpenAI, or Anthropic
- **SM-2 spaced repetition**: Efficient long-term retention with flashcard study mode
- **Cross-browser**: Chrome, Firefox, and Edge support

## Quick Start

```bash
npm install
npm run build:chrome   # or build:firefox / build:edge
```

Load the `dist/` directory in your browser's extension page.

## Build

| Browser   | Command                  |
|-----------|--------------------------|
| Chrome    | `npm run build:chrome`   |
| Firefox   | `npm run build:firefox`  |
| Edge      | `npm run build:edge`     |
